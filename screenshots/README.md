This directory contains screenshots of the Blood Donor Management System user interface.

The following screenshots should be added to compile the report.tex PDF successfully:

1. **login-registration.png** - Login and Registration Page with OTP verification
2. **donor-dashboard.png** - Donor Dashboard showing profile summary

4. **emergency-request-form.png** - Emergency Blood Request Form

6. **donor-profile-management.png** - Donor Profile Edit Interface

## How to Generate Screenshots

You can take screenshots from the live application:
**Live URL:** https://blood-donor-management.vercel.app

### Recommended Screenshot Specifications
- Format: PNG
- Dimensions: 1920x1080 or higher
- Aspect ratio: 16:9
- File size: < 1MB per screenshot

### Screenshot Guidelines
1. Use a clean browser window (no extensions/bookmarks visible)
2. Ensure good contrast and readability
3. Include representative data (not production data)
4. Capture full feature context
5. Use consistent zoom level across all screenshots

## Alternative: Placeholder Images

If actual screenshots are not available, you can temporarily use placeholder images:
- Create blank PNG files with the correct dimensions
- Or use placeholder image services
- Update with actual screenshots before final submission
